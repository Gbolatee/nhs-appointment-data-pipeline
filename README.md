# NHS Appointment Data Engineering Pipeline

## Overview

This project demonstrates an end-to-end data engineering pipeline built using NHS open data. The pipeline extracts appointment data, performs data cleaning and validation, loads the processed data into PostgreSQL, and produces analytical dashboards in Power BI.

The project is designed to showcase practical data engineering skills relevant to NHS data engineering and business intelligence roles.

## Objectives

- Build a robust ETL pipeline using Python
- Clean and validate NHS appointment data
- Store processed data in PostgreSQL
- Perform analysis using SQL
- Create interactive Power BI dashboards
- Apply Git and GitHub version control
- Document the complete data engineering workflow
